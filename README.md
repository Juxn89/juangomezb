# Commands
## Yarn
- **Restore package:** ```yarn rebuild```
- **Run project | development:** ```yarn dev```
- **Build:** ```yarn build```
- **Start project | producction:** ```yarn start```

## NextJS
- **Export:** ```yarn run next:export```

## ESLint
- **Run ESLint**: ```yarn run lint```
- **Fix**: ```yarn run lint --fix```
